# 🚀 快速启动（3步搞定）

## 第1步：安装 Docker Desktop

**Windows用户**：
1. 下载：https://www.docker.com/products/docker-desktop
2. 安装后启动 Docker Desktop
3. 等待它完全启动（右下角图标变绿）

**验证安装**：
```bash
docker --version
```

---

## 第2步：获取API密钥（选一个就行）

### 推荐：Gemini（免费，最简单）

1. 打开：https://aistudio.google.com/app/apikey
2. 登录你的Google账号
3. 点击"创建API密钥"
4. 复制那个密钥（类似：`AIzaSy...`）

---

## 第3步：配置并运行

### 3.1 创建配置文件

在项目根目录创建 `backend/.env` 文件，内容如下：

```env
GEMINI_API_KEY=你的API密钥粘贴这里
POSTGRES_PASSWORD=123456
SECRET_KEY=随便写一个32位以上的字符串比如abcdefghijklmnopqrstuvwxyz123456
```

**注意**：把 `你的API密钥粘贴这里` 替换成你刚才复制的Gemini密钥

### 3.2 运行项目

打开命令行（PowerShell或CMD），在项目根目录执行：

```bash
cd infrastructure/docker
docker-compose up -d
```

**等30秒左右**，然后访问：http://localhost:8000/docs

---

## ✅ 完成！

如果看到API文档页面，说明运行成功了！

---

## 🛑 停止项目

```bash
cd infrastructure/docker
docker-compose down
```

---

## ❓ 如果出错了

**查看日志**：
```bash
cd infrastructure/docker
docker-compose logs
```

**常见问题**：
- Docker没启动？→ 打开Docker Desktop
- 端口被占用？→ 关闭占用8000端口的程序
- API密钥错误？→ 检查 `.env` 文件中的密钥是否正确

---
